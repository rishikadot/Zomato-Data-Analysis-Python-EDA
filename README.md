# Zomato Restaurant Data Analysis | Python EDA Project

## Objective
Analyze Zomato restaurant dataset to understand consumer behavior, rating patterns and online vs offline ordering preference.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Steps Performed
- Imported & cleaned dataset
- Converted inconsistent rating format (e.g. "4.1/5" → 4.1 float)
- Performed Exploratory Data Analysis (EDA)
- Visualized patterns using histograms, bar charts and heatmap
- Derived business insights from the dataset

## Key Insights
- Dining restaurants are the most common category.
- Restaurants offering online orders tend to receive higher ratings compared to offline-only restaurants.
- Cafés show higher online ordering behavior while Dining restaurants receive more offline orders.
- Majority of restaurants have ratings between **3.5 to 4.0**.
- Premium cost restaurants generally have slightly better ratings.

## Dataset Columns
- name  
- online_order  
- book_table  
- rate  
- votes  
- approx_cost(for two people)  
- listed_in(type)

## Conclusion
This analysis shows how online convenience, restaurant type and pricing influence customer engagement and ratings in the restaurant industry.
